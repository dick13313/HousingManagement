# 房租管理系統

以屋主收租管理為核心的 Nuxt 3 + Supabase 專案。

## 技術方案

- Nuxt 3
- Supabase
- Netlify Free

完整方案請看 [docs/solution.md](docs/solution.md)。

## 開發

建立 `.env` 並填入 Supabase 設定：

```bash
SUPABASE_URL=
SUPABASE_KEY=
```

安裝依賴並啟動：

```bash
npm install
npm run dev
```
